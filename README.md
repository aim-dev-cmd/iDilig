# iDilig
A smart soil monitoring platform for pechay farming with real-time data, alerts, and analytics.

---

iDilig is an innovative platform designed to improve soil monitoring for optimal pechay (Chinese cabbage) cultivation. Established in 2023, it helps farmers and gardening enthusiasts monitor soil conditions by providing real-time data on moisture and temperature. The system supports precision farming and helps improve crop yield through better decision-making.

Features

User Registration
Secure access with a personalized experience.

Real-Time Monitoring
View live soil moisture and temperature data.

Historical Trends
Analyze past data to support better farming decisions.

Alert System
Receive notifications when soil conditions are not ideal.

Dashboard
Simple and easy-to-use interface for monitoring data.

How It Works

Users register for an account and log in to access the system.
They can monitor real-time soil data through the dashboard.
The system provides alerts and insights to help manage irrigation and crop care effectively.


Purpose

The goal of iDilig is to improve farming efficiency, reduce resource waste, increase crop yield, and promote smart agriculture practices.
